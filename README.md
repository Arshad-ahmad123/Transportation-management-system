# Transportation-management-system

This repository contains a complete **Transportation Management System (TMS)** built by **Arshad Ahmad Amiri**, **Muhammad Akram Mal**, and **Muhammad Rafi Tahiri**.  
The project is designed for transportation companies to efficiently manage and store all essential operational data in a structured and scalable database system.

## ğŸ’» Installation & Setup

To run this project on your PC:

### 1. Install Microsoft SQL Server
Required to host and manage the database structure, tables, and data.

### 2. Restore or Import the Database
Use **SQL Server Management Studio (SSMS)** to open and execute the provided `.sql` file.

### 3. Setup Web Interface
Configure your PHP database connection file with:
- SQL Server host  
- Database name  
- Username  
- Password  

### 4. Run the System
Open your browser and access the local PHP interface to interact with the database.
